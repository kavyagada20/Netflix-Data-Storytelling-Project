# 🎬 Netflix Data Storytelling Project

## 📌 Objective

The goal of this project is to analyze Netflix's content dataset and uncover meaningful insights about its content strategy, growth trends, and global distribution using data analytics.

---

## 📂 Dataset

* Source: Kaggle - Netflix Movies and TV Shows Dataset
* Total Records: 8807
* Features: Title, Type, Country, Genre, Date Added, Duration, etc.

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* (Optional) SQL

---

## 📊 Analysis Performed

### 1. Movies vs TV Shows

* Compared distribution of content types on Netflix.

### 2. Country-wise Analysis

* Identified top content-producing countries.

### 3. Year-wise Growth

* Analyzed how Netflix content has grown over time.

### 4. Genre Analysis

* Identified most popular genres on Netflix.

## 💡 Key Insights

* Netflix has significantly more **Movies** than TV Shows.
* Content production increased rapidly after **2016**, indicating aggressive expansion.
* The **United States dominates** content production.
* **India is emerging** as a major contributor.
* **Drama and Comedy** are the most popular genres globally.
* Rise in TV Shows suggests a shift toward binge-watching culture.
